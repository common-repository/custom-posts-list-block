Provides a Gutenberg Block to list posts based on the selected category.

== Description ==
This custom Gutenberg block is used for adding a posts list based on the default post category. Make sure you have categories added

#### FEATURES
- You can create a custom posts list by selecting a category.
- Show or hide Category names, post images, and post excerpts based on the options selected.

== Installation ==
Upload the plugin to the “/wp-content/plugins/” directory.
Activate the plugin through the “Plugins” menu in WordPress.
